Notes for the Macintosh 68k implementation
------------------------------------------
- the Macintosh 68k implementation is provided as a CodeWarrior project.
- the kernel specific implementation (as defined in 'msExtern.h') is located
  in the 'Implementation Mac68k' directory.
- other files only implements the MidiShare loader.
- the current output of the project is a Macintosh application.

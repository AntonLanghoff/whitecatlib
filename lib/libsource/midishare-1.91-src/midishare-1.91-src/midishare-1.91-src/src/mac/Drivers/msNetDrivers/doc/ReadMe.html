MidiShare Internet Drivers

version française

The MidiShare Internet Drivers provides real-time communication over Internet to MidiShare client applications.
These drivers are two separate applications:

  • msLAN : provides low latency real-time communication restricted to local array networks. It includes automatic recognition of remote hosts.
  • msWAN : provides real-time communication over Internet. Remote host connection is controlled by the user.

Both drivers have the following common characteristics:

  • remote hosts appears as MidiShare ports or as MidiShare clients, depending on the driver running mode.
  • as remote clients provides a unique identifier with their IP address, their connections state is saved and automatically restored using this identifier. Depending on the driver running mode, these connections may be application connections or ports connections.

The Setup menu:

  • Driver mode : when checked, the application is running as a MidiShare driver and the remote hosts appear as MidiShare ports. Otherwise it runs as a MidiShare application and the remote hosts appear as MidiShare client applications.
  • Hide window: show or hide the application window
  • Auto quit: when checked, the application automatically quit when there are no more MidiShare clients.
  • UDP preferences: allows to control the UDP transmission parameters.

The MidiShare WAN driver is the base of the Radio Hard Disk project which notably consists in a permanent music broadcasting over Internet.

The MidiShare Internet Drivers are part of the MidiShare project. They have been developped by the Grame's Computer Music Research Laboratory, in collaboration with the company Mil-Productions.


msplayer is a MIDIFile player that use the MidiShare Player library

To compile msplayer you'll need to compile and install the MidiShare Player library first

The Player library can be downloaded at the following location : 

ftp://www.grame.fr/pub/midishare-lib/player-linux.tar.gz

or directly from the CVS server at : http://www.grame.fr/MidiShare/SCPP/dev.html#lib

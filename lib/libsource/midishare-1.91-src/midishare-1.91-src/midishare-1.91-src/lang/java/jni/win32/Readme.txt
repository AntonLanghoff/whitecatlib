MidiShare native code interface library
----------------------------------------------------------------
How to compile the JMidi.dll
----------------------------------------------------------------

The project provided with this source code distribution is 
a Visual C++ 5.0 project (JMidi.dsp).
To compile, you need to provide two additionnal interface files:
	- jni.h
	- jni_md.h
These files contains standard java native interface declarations.
They should be part of any Java Development Kit. You should copy 
them to the project folder.

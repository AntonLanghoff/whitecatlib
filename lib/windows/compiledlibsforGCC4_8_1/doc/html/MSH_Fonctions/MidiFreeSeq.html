MidiFreeSeq

Frees a sequence and its content. MidiFreeSeq first de-allocates all the events of the given sequence and then the sequence header itself. 

pascal void    MidiFreeSeq (MidiSeqPtr s);

s
a MidiSeqPtr, is a pointer on a sequence to be freed.

Frees a previously allocated sequence s. 


MidiSeqPtr    s;

s = MidiNewSeq();

/* ....... */

MidiFreeSeq(s);

Once freed, s is no longer a valid pointer.

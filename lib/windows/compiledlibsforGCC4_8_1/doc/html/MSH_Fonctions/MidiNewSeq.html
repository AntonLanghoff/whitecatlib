MidiNewSeq

Allocation of a new empty sequence. 

pascal MidiSeqPtr    MidiNewSeq();

none

The result is a MidiSeqPtr, a pointer to an empty sequence.

Create a sequence of 10 Midi clocks. 


MidiSeqPtr ClockSeq()
{
    MidiSeqPtr    s;
    MidiEvPtr    e;
    long    d;
    
    s = MidiNewSeq();
    for (d=0; d< 2500; d+=250) 
    {
        e = MidiNewEv (typeClock);
        Date(e) = d;
        MidiAddSeq (s, e);
    }
    return s;
}

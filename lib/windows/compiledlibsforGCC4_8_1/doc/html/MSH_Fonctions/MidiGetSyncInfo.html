MidiGetSyncInfo

Fills a TSyncInfo record with information about the current state of the MTC synchronisation. 

pascal void MidiGetSyncInfo (SyncInfoPtr p);

p
a SyncInfoPtr, a pointer to a TSyncInfo record.

Gives the SMPTE start location of the tape. 


TSyncInfo     myInfo;
TSmpteLocation    myLoc;
MidiGetSyncInfo(&myInfo);
MidiTime2Smpte( MidiInt2ExtTime(myInfo.syncStart), myInfo.syncFormat, &myLoc);

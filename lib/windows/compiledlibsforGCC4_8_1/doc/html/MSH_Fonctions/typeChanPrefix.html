typeChanPrefix (code 142)

A channel prefix event (from the Midi File 1.0 specification). This event cannot be sent to external Midi devices.

typeChanPrefix events have one field.

0
A channel prefix number from 0 to 15. (Field size : 1 byte)

Creates a typeChanPrefix event. Return a pointer to the event or NIL if there is not enough memory space. 


MidiEvPtr ChanPrefix ( long date, short prefix)
{
    MidiEvPtr e;
    long    c=0;

    if ( e = MidiNewEv(typeChanPrefix))    /* Allocate a new event. Check not NIL */ 
    {
        Date(e) = date;                    
        MidiSetField( e, 0, prefix);
    }
    return e;
}

Content of the msNotes archive
==============================
Content of this archive are the following  folders:
  Documentation - Development - Executable and Tutorial.

To use it you need to do these steps:
    (1) create a parent folder ( for example: /msNotes ).
    (2) extract all the content of the archive in this parent folder.
    (3) Choose a documentation to read (printable or on-line hypertext).

The description of each folder follow.

1. folder: Documentation
------------------------
This folder is the printable and on line documentation.

MsNotes102Fr.doc	Documentation (French) (Microsoft Word 97).
MsNotes102Fr.pdf	Documentation (French) (Adobe Acrobat Reader).
MsNotes102Eng.doc	Documentation (English) (Microsoft Word 97).
MsNotes102Eng.pdf	Documentation (English) (Adobe Acrobat Reader).

One line specific documentation for Windows is in the Documentation\WinHelp folder. 
Open the msNotes.hlp help file.

One line cross - platform documentation  is in the Documentation\WebHelp folder. 
Open the msNotes.htm file with your favorite browser.

2. folder: Development
----------------------
These files are used during the  development of a MidiShare application using the
MsNotes library.
See content.txt file.

3. folder: Executable
---------------------
The file of this folder are used only during execution of an  application using the
MsNotes library.
See content.txt file.

4. folder: Tutorial
-------------------
These files are the sources files of the tutorial program tutorial.exe. 
See content.txt file.
